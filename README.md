Assignment 4 - Python File Handling

Description

This assignment demonstrates the use of Python file handling concepts including:

1. reading files
2. writing files
3. appending data to files
4. processing file data
5. basic error handling

How to Run

1. Open VS Code or terminal
2. Navigate to the project folder
3. Run each file separately:

python task1_sales_records.py
python task2_read_file.py
python task3_append_sales.py
python task4_summary_report.py
python task5_product_file.py
python task6_safe_read.py
python task7_discount_project.py
